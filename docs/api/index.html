ICU for Haxe

intl version 0.9.1

Provides a small subset of the International Components for Unicode (ICU), for Haxe.

Collator

Provides string comparison capability with support for appropriate locale-sensitive sort orderings.

CollatorOptions

Defines the options of a Collator instance.

CollatorStrength

Specifies the strength of a collator.

DateFormat

Formats dates in a locale-dependent manner.

DateFormatOptions

Defines the options of a DateFormat instance.

DateFormatStyle

Specifies the formatting style of a date or time.

DateFormatTools

Provides static extensions for dates.

Locale

Represents a Unicode identifier used to get language, culture, or regionally-specific behavior.

NumberFormat

Formats numbers in a locale-dependent manner.

NumberFormatOptions

Defines the options of a NumberFormat instance.

NumberFormatStyle

Specifies the formatting style of a number.

NumberFormatTools

Provides static extensions for numbers.

NumberFormatUnitDisplay

Defines the unit formatting style.

Region

Represents a country/region.

SimpleUnit

Provides the list of supported simple units.

TimeZone

Represents a time zone offset, and also figures out daylight savings.

TimeZoneOptions

Defines the options of a TimeZone instance.

TimeZoneStyle

Specifies the formatting style of a time zone.
